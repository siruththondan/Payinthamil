Each colour = one finger. Rest on home row (க ப ம த ந ய) and reach from there.

Home-row keys need no finger movement. Reach all other keys from that resting position.

Setup Guide — Tamil99 Keyboard
⌨️
Use English keyboard layout

This app uses the Tamil99 layout on top of a standard English (QWERTY) keyboard. Make sure your OS input is set to English before typing — Tamil output happens automatically via the Tamil99 key mapping.

Win Space Switch input on Windows
⌘ Space Switch input on macOS
Super Space Switch input on Linux
1
Download Tamil99 Keyboard Layout

Go to the Tamil Virtual Academy website and download the official Tamil99 keyboard driver for Windows.

tamilvu.org — Tamil99 Download

Alternatively, download from NHM Writer (free, lightweight) which includes Tamil99 support.

NHM Tamil Writer
2
Install the driver

Run the installer as administrator. It adds a new keyboard layout to Windows.

3
Add Tamil99 input to Windows

Go to Settings → Time & Language → Language & Region. Click Add a language, search for Tamil, and add it. Then click the Tamil language → Options → Add a keyboard → select Tamil99.

4
Switch between layouts

Use Win + Space to toggle between English and Tamil99. The taskbar language indicator shows the current layout.

💡 For this website, keep your layout set to English — the Tamil99 mapping is handled in the browser.
1
Tamil99 is built into macOS

No download needed. macOS includes Tamil99 as a built-in input source.

2
Add Tamil99 input source

Go to System Settings → Keyboard → Input Sources → Edit. Click +, search for Tamil, and select Tamil99. Click Add.

3
Show the input menu in menu bar

Enable Show Input menu in menu bar in the same Keyboard settings. This lets you quickly see and switch the active layout.

4
Switch between layouts

Use ⌘ + Space (or the menu bar icon) to toggle between English and Tamil99.

💡 For this website, keep your layout on English (ABC) — Tamil output happens automatically.
1
Install Keyman (recommended — official Tamil99 support)

Keyman is the easiest way to get Tamil99 on Linux with full Unicode support.

sudo add-apt-repository ppa:keymanapp/keyman
sudo apt update
sudo apt install keyman ibus-keyman

On Fedora: sudo dnf install keyman ibus-keyman. On Arch: yay -S keyman. Or download from keyman.com/linux.

2
Install the Tamil99 keyboard package

Open the Keyman Configuration app → Download keyboards → search Tamil99 → install Tamil99 (tam99).

Or install from terminal: sudo keyman install tam99

3
Alternative: IBus + m17n (lighter option)

If you prefer IBus without Keyman:

sudo apt install ibus ibus-m17n m17n-db
ibus restart

Then go to IBus Preferences → Input Method → + → search Tamil → select ta - tam99 (m17n).

GNOME users: Settings → Keyboard → Input Sources → + → Other → Tamil → Tamil (tam99).

4
Set IBus as default input method (if needed)

Add to ~/.profile or ~/.bashrc:

export GTK_IM_MODULE=ibus
export QT_IM_MODULE=ibus
export XMODIFIERS=@im=ibus

Log out and back in. Use Super + Space to switch layouts.

💡 For this website, keep your OS layout on English — Tamil output is handled by the browser. You only need Tamil99 installed for typing in other apps.
